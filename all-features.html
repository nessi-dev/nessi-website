All Features

Comprehensive overview of all Nessi.dev capabilities

Complete Feature Set

Discover the full range of tools and capabilities in Nessi.dev

Full Schema Management

Schema evolution tracking and history, validation on write operations, field-level metadata exploration, partition intelligence with Z-ordering awareness, and metadata inspection with optimization hints.

Version Control

Transaction history with commit details, data rollback capabilities (up to 30 days), and version comparison with change summaries.

Time Travel

Point-in-time querying (up to 30 days history), timestamp and version-based access, and historical state reconstruction.

Automated Profiling

Statistical summaries (min, max, mean, median), distribution analysis with histograms, null percentage tracking, and outlier identification.

Data Lineage Visualization and Comparison

Interactive lineage graphs, dependency tracking, impact analysis for schema changes, and data flow visualization with transformation details.

CLI Reports

Quality score cards with drill-down capabilities, distribution histograms and visualizations, HTML/PDF reports with shareable links, and custom report templates.

Alerting

Email notifications for critical issues, threshold-based triggers, daily/weekly summary reports, and customizable alert templates.

Cloud Integration

AWS S3 integration with IAM authentication, Azure Blob Storage with Azure AD, GCP Cloud Storage with service accounts, and cross-cloud Delta Lake management.

Security

SSL/TLS encryption for all communications, JWT authentication for API access, API key management for programmatic access, and role-based access control for features.

Data Catalog Integration

AWS Glue Data Catalog integration, Azure Purview integration, Google Cloud Data Catalog integration, data asset discovery and connection, metadata utilization, and quality metrics and lineage publishing back to catalogs.

Pattern Recognition

Regex pattern extraction and matching, common format validation (emails, dates, IDs), and data format consistency monitoring.

CLI Monitoring

Table health scores and trends via CLI commands, quality metric tracking over time with exportable data, recent validation failures reporting, and table health and ingestion metrics via CLI.

Monitoring History

30-day metrics retention, trend visualization and comparison, and performance impact analysis.

Collaboration & Governance

Shared rule libraries and configurations.
