Powerful Features

Essential tools for exceptional Delta Lake data quality and management

Full Schema Management

Schema evolution tracking and history, validation on write operations, field-level metadata exploration, partition intelligence with Z-ordering awareness, and metadata inspection with optimization hints.

Version Control

Transaction history with commit details, data rollback capabilities (up to 30 days), and version comparison with change summaries.

Time Travel

Point-in-time querying (up to 30 days history), timestamp and version-based access, and historical state reconstruction.

Automated Profiling

Statistical summaries (min, max, mean, median), distribution analysis with histograms, null percentage and unique value ratios, type inference and consistency checks, and completeness, accuracy, and consistency scoring.

Anomaly Detection

Outlier detection using z-score and IQR methods, sudden change detection between runs, constant changes detection, sudden spikes/dips detection, oscillations detection between runs, and historical trend deviation alerts.

Data Lineage Visualization

Interactive lineage graphs, dependency tracking, impact analysis for schema changes, and data flow visualization with transformation details.

Alerting System

Email notifications for critical issues, threshold-based triggers, daily/weekly summary reports, and customizable alert templates.

Cloud Integration

AWS S3 integration with IAM authentication, Azure Blob Storage with Azure AD, GCP Cloud Storage with service accounts, and cross-cloud Delta Lake management.

Seamless Integrations

Harmoniously connect with your existing data ecosystem

Databricks Integration

Seamlessly work with Databricks Unity Catalog and Delta Lake tables for comprehensive data quality management.

dbt Integration

Opt-in dbt plugin for data quality and profiling, execute Nessi.dev data quality rules against dbt models, enhanced model selection with dbt syntax, generate data profiles for Delta tables associated with dbt models, and automatic mapping between dbt models and Delta tables.

Cloud Storage Compatibility

Seamless integration with Delta Lake on AWS S3, Azure Blob Storage, and GCP Cloud Storage for multi-cloud management.

Workflow Orchestration Integration

Apache Airflow operators and sensors for Nessi.dev tasks, Prefect tasks and flows integration, Dagster ops and resources integration, Kubernetes operators for Nessi.dev jobs, workflow status monitoring and callbacks, and pipeline-aware data quality checks.
