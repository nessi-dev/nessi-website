Powerful Features

Essential tools for exceptional Delta Lake data quality and management

Full Schema Management

Schema evolution tracking and history, validation on write operations, field-level metadata exploration, partition intelligence with Z-ordering awareness, and metadata inspection with optimization hints.

Version Control & Time Travel

Transaction history with commit details, data rollback capabilities (up to 30 days), version comparison with change summaries, and timestamp/version-based point-in-time querying and historical state reconstruction.

Automated Data Profiling

Statistical summaries (min, max, mean, median), distribution analysis with histograms, null percentage and unique value ratios, type inference and consistency checks, and completeness, accuracy, and consistency scoring.

Anomaly Detection

Outlier detection using z-score and IQR methods, sudden change detection between runs, constant changes detection, sudden spikes/dips detection, oscillations detection between runs, and historical trend deviation alerts.

Data Lineage Visualization

Interactive lineage graph visualization, upstream and downstream dependency tracking, version comparison of lineage changes, impact analysis for schema changes, time-based lineage evolution view, and export to various formats.

Rule Validation

Predefined rules library (null checks, range validation, length, regex, enum), custom rule creation via YAML/SQL/Python, rule execution history and impact analysis, and row-level rule validation with custom logic.

CLI Reports

Quality score summaries via CLI, table and column level summaries, trend data for external visualization, distribution data export, pattern frequency analysis, failure rate tracking, export to JSON/CSV for automation and external reporting, and machine-readable output for integration with other tools.

Alerting System

Email notifications for critical issues, threshold-based triggers, and daily/weekly summary reports with customizable templates.

Cloud Integration

AWS S3 integration for Delta Lake tables, Azure Blob Storage integration with latest SDK, GCP Cloud Storage integration, multi-cloud Delta Lake management, cloud-native authentication methods, and cross-cloud data quality validation.

Developer Experience

CLI-first with one-line validation commands, batch processing scripts, configuration management, Python API for programmatic access, webhook integration, custom extension support, embeddable in Airflow, GitHub Actions, or custom scripts, interactive examples, API reference, best practices guides, ASCII schema trees and CLI heatmaps, Terraform-style CLI flags, local-only data handling for privacy, and GitHub usage badge.

Governance & Security

Shared rule libraries and configurations, secure local file operations, environment variable-based authentication for cloud services, API key management for external services, and basic access controls for CLI commands.
