Nessi

The local-first toolkit for understanding, validating, and explaining your Delta Lake tables.

Delta Lake Health, Simplified

Table Format Detection: Automatically detects Delta Lake, Parquet, or CSV tables with intelligent fallback logic and clear error messages.

Historical Statistics: Analyze commit history, partition change patterns, and data evolution trends across versions.

Sample Diffing: Export changed rows between table versions to CSV/Parquet — great for audits and debugging.

Built for Data Engineers

Nessi integrates into your CLI, CI/CD pipelines, or Python scripts. With a blazing-fast engine powered by Spark and pandas, it handles massive datasets with intelligent defaults, caching, and partition pruning.

Visual, Configurable Reporting

HTML Output: One command generates beautiful, professional reports with summary cards, partition heatmaps, and validation outcomes.

Validation Rules: Configure rules like max-null-percent or min-unique-pct, with results categorized as PASS, WARN, FAIL, or INFO.

Run Anywhere. Own Everything.

Nessi runs 100% locally — no SaaS, no data sharing. Use it offline, in secured environments, or plug it into AWS, Azure, or GCP.
