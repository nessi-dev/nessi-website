Powerful Features

Essential tools for exceptional Delta Lake data quality and management

Full Schema Management

Schema evolution tracking, validation on write, field-level metadata exploration, and partition intelligence with Z-ordering awareness.

Version Control & Time Travel

Transaction history with commit details, data rollback capabilities (up to 30 days), and timestamp-based point-in-time querying.

Automated Data Profiling

Statistical summaries (min, max, mean, median), distribution analysis with histograms, and null percentage tracking.

Anomaly Detection

Outlier detection using z-score and IQR methods, sudden change detection, and oscillations detection between runs.

Data Lineage Visualization

Interactive lineage graph visualization, upstream/downstream dependency tracking, and impact analysis for schema changes.

Monitoring Dashboard

Built-in dashboard for core monitoring functionality, real-time metrics visualization, and system health monitoring.

Rule Validation

Predefined rules library (null checks, range validation, regex), custom rule creation via YAML/SQL/Python, and rule execution history.

Interactive Reports

Quality score cards with drill-down, distribution histograms, and HTML/PDF reports with shareable links and expiration.

Alerting System

Email notifications for critical issues, threshold-based triggers, and daily/weekly summary reports with customizable templates.

Cloud Integration

AWS S3, Azure Blob Storage, and GCP Cloud Storage integration with cloud-native authentication methods for Delta Lake tables.

Developer Experience

CLI-first with one-line validation commands, Python API, and embeddable in Airflow, GitHub Actions, or custom scripts.

Governance & Security

Shared rule libraries, SSL/TLS encrypted interfaces, user authentication with JWT, and API key management.
