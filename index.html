Nessi

The local-first toolkit for understanding, validating, and explaining your Delta Lake tables.

Get Started

Delta Lake Health, Simplified

Table Format Detection: Automatically detects Delta Lake, Parquet, or CSV tables with intelligent fallback logic and clear error messages.

Historical Statistics: Analyze commit history, partition change patterns, and data evolution trends across versions.

Sample Diffing: Export changed rows between table versions to CSV/Parquet — great for audits and debugging.

Built for Data Engineers

Nessi integrates into your CLI, CI/CD pipelines, or Python scripts. With a blazing-fast engine powered by Spark and pandas, it handles massive datasets with intelligent defaults, caching, and partition pruning. Easily deploy using Docker or Docker Compose in local and team environments.

Visual, Configurable Reporting

HTML Output: One command generates beautiful, professional reports with summary cards, partition heatmaps, and validation outcomes.

Validation Rules: Configure rules like max-null-percent or min-unique-pct, with results categorized as PASS, WARN, FAIL, or INFO.

Advanced Profiling: Nessi calculates min/max, mean, median, standard deviation, and cardinality for numeric and categorical columns.

Custom Sampling: Easily export samples of changed rows with custom sizes and formats using CLI flags like --dump-sample.

Plugin-Friendly: Extend Nessi with your own validation logic or exporters using a simple plugin architecture.

Validation That Goes Deeper

Comprehensive Validation Checks: Automatically verify row counts, detect duplicates, flag out-of-range values, and catch type mismatches or schema drift before they hit production.

Custom Validation Scripting: Define your own validation logic using lightweight rule scripts or plugins to enforce data contracts unique to your domain.

Run Anywhere. Own Everything.

Nessi runs 100% locally — no SaaS, no data sharing. Use it offline, in secured environments, or plug it into AWS, Azure, or GCP.

Yes! Nessi is designed for automation and works with GitHub Actions, Airflow, and other orchestration tools.
